# 上条剛寛 — ポートフォリオ

このリポジトリは、上条剛寛さんの個人ポートフォリオサイトのシンプルな静的テンプレートです。

主な経歴・強み:
- IT系新規事業企画のプロジェクト・プロダクトマネージャー経験
- 大手総合商社における社内新規企画の立案経験が多数
- 得意領域：サプライチェーンの最適化におけるデータ利活用（需要予測、在庫最適化、ダッシュボード設計）

## 同梱ファイル
- `index.html` — メインのポートフォリオページ
- `styles.css` — サイトのスタイル
- `script.js` — ナビ・フォームの簡易スクリプト
- `README.md` — このファイル

## ローカルでの確認方法（Windows）
1. このリポジトリのフォルダを任意の場所にクローンまたはコピーする。
2. ブラウザで `index.html` を開く（エクスプローラーでファイルをダブルクリック）。

より正確にローカルサーバで確認したい場合（推奨）:
```powershell
# PowerShell で簡易サーバを起動する例 (Python が必要)
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## フォームと連絡先の設定
現在フォームはダミーの `action="#"` になっています。Formspree などを使う場合は、取得したエンドポイントを `index.html` の `<form action="...">` に設定してください。

## デプロイ候補
- GitHub Pages、Netlify、Vercel などの静的ホスティングサービスにそのままデプロイできます。

## カスタマイズ可能な項目
- 表示名・職種・実績の文章は `index.html` を編集してください。
- 画像は `https://via.placeholder...` の箇所に差し替えてください。
- 連絡先（メール等）は `index.html` のお問い合わせ欄で置き換えてください。
